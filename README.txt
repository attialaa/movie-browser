Movie Browser - Ready project
-------------------------------

Steps to run:
1. Move the zip contents to your Desktop or preferred folder.
2. Open terminal and cd into the folder, e.g. `cd ~/Desktop/movie-browser`
3. Install dependencies: `npm install`
4. Replace YOUR_API_KEY_HERE in App.js with your TMDb API key.
5. Start Expo: `npx expo start`

If expo warns about CLI, use `npx expo start`. Use Expo Go app to scan QR code.
